# JASSO GPA Calculator

A simple, unofficial tool for calculating GPA using JASSO's grade-point conversion
formula (5-level scale: A/B/C/D/F). For anyone looking to calculate their GPA for
a Japan scholarship application under JASSO's criteria, feel free to use this as
a rough reference tool for your calculation.

**Note:** This is not an official JASSO tool. Always double-check against your
program's official announcement for the exact minimum GPA required.

## Usage

Open `index.html` in a browser or (https://claude.ai/public/artifacts/628d0ba3-651b-4dcd-bf50-6928e320b44f).

- Paste grades separated by spaces or commas (e.g. `A B+ C- D A- A B C+ F`)
- Adjust credits per course if needed
- GPA updates automatically, with a pass/fail indicator against the 2.30 minimum
